# Typlux

Typlux is a TypeScript friendly implementation of Flux architecture which is writtern in TypeScript.

## Install

* npm

```
$ npm install typlux
```

* yarn

```
$ yarn add typlux
```

## Example

[Simple Todo Application](https://github.com/niba1122/typlux-example)

## Development

```
$ npm install
$ npm run dev
$ npm run build
```
